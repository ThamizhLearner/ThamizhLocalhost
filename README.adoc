To quickly try out the Thamizh Localhost server use the following command +
`go run github.com/ThamizhLearner/ThamizhLocalhost@latest`

Ensure you have https://go.dev/dl/[go-lang] installed beforehand for the command to work.

Open the browser and type `localhost:8080` in the address bar to access the local server.
